SQL INSERT Statement: Adding Data to Tables
The INSERT statement is used to add new rows to a table. There are two main ways to use INSERT:

4Ô∏è‚É£ Using DEFAULT Values

üìå If department has a default value ('General'), it will be used.

Using DEFAULT Values in INSERT Statement
The DEFAULT keyword is used in SQL INSERT statements when you want a column to use its predefined default value instead of manually inserting a value.

1Ô∏è‚É£ Basic INSERT Using DEFAULT

VALUES (value1, DEFAULT, value3);
üîπ The DEFAULT keyword allows the column to take its predefined value.

2Ô∏è‚É£ Example Table: employees

    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT DEFAULT 25,
    department VARCHAR(50) DEFAULT 'General',
    salary DECIMAL(10,2) DEFAULT 40000
);
üìå Defaults:

age ‚Üí 25
department ‚Üí 'General'
salary ‚Üí 40000
3Ô∏è‚É£ Inserting Data with DEFAULT

VALUES (101, 'Alice', DEFAULT);
üîπ Since department has a default ('General'), it will be inserted as 'General'.

4Ô∏è‚É£ Using DEFAULT for Multiple Columns

VALUES (102, 'Bob', DEFAULT, DEFAULT, 55000);
üîπ age = 25, department = 'General', salary = 55000.

5Ô∏è‚É£ Inserting Only DEFAULT Values
